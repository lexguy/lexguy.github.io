时间很短
我只是不想下次回顾的时候，像今天忆起过往一样难过

你好，我是 Xuwei。

一个普通的前端开发者，目前是 React && React Native 方向。想不出什么其他闪光点，或许这正是当下的我。

关于 xuwei.plus

偶尔会写些东西，在本地有道云笔记，或者发在 CSDN，掘金。虽然更新频率不高，但有时产生写作冲动时，总觉得要有个更加自由的平台才舒服，毕竟想写也不总是技术相关。这个想法之后，再往其他平台上面写东西就越来越觉得有建立自由领地的必要。有了想法之后就得行动，无奈在选择技术和方案的时候，一直犹豫，再加上本来不足的时间偶尔被挥霍，就拖到了现在。最后还是毫无技术地在 hexo 上选了个主题开工。时间果真是耗不起的，现在就开始，做起来，比什么都强。

记录 or 分享

有时候写博客会犯难，以一种什么样的出发点和角度来发布。 通常对于个人笔记来说，更重要的是记录，结合实例场景描述一切从简，但是在可读性上更多倾向面对自己，其他人可能很难读懂具体场景和内容是什么，又或者读懂了也获得不了多大价值。而区别于个人笔记，博客应该更多的是分享，要面对更广的受众而非自己。所以要是一些具有普遍价值，带有分享性质的内容，才应该置于博客平台。所以，这里除开碎碎闲谈，技术上的内容，力求分享和引申出其他价值。

闲谈网站其他

网站的首页，顶部图是早年的新浪博客的一个主题图。新浪博客是我的第一个博客，当时读初三，年少的自己喜欢许嵩，多少受一些影响。本来中二时期就喜欢装 13，还是关于抒情文艺的那种，就跟着用上了这个名为 [雾林] 的主题，还喜欢写一些或世俗或愤青的文字在上面。

建立 xuwei.plus 的时候想起来那个新浪博客，试了下居然还能登上去，虽然有满满的羞耻，但是立马截取了这个主题大图，换做现在博客的大图，虽然不全是“当年”的感觉，但感受到一种完整和得以延续的踏实，也算是中二时期留下的一份礼物吧。

在自己的领地写东西，会有一种不一样的自由。区别于私有的有道云笔记的本地写作，也区别于各个带有明显内容倾向的博客平台。可以写技术上的需要记录的重点，新内容的阶段性的成果，也可以写日常生活，所想所得。这里写东西，自己一面闲暇记录的同时，也试着输出一些引人注目的干货。

像个街角唱台的艺人，无人叫好的时候，得更用心的唱。
